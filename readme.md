# SecNews

安全资讯平台，为甲方安全部添砖加瓦，升职加薪全靠它了

## 功能列表

- 爬虫功能：爬取常见的漏洞资讯
- 爬虫定时任务设置：指定定时任务在线配置
- 资讯推荐功能：从每日的大量资讯中推送公司资产相关的漏洞资讯，集中精力办大事
- 订阅功能：支持邮箱订阅
